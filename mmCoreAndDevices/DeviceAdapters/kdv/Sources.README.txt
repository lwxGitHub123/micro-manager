These are driver sources to use the focus drive for Meiji Techno stereo microscopes in micro-manager.
Compiled with MMDeviceKit-win-34-01-20100119
Tested with micro-manager 1.3.47 on Windows XP.
Provided "as is".

Compilation instructions on Windows:
- Install micro-manager 1.3.47 (or later)
- Install micro-manager device driver kit MMDeviceKit-win-34-01-20100119.zip (or later)
- unzip KDV-zstage-sources.zip sources to C:\Program Files\Micro-Manager-1.3\MMDeviceKit-win-34-01\DeviceAdapters\KDV
- compile project KDV.vcproj
- copy mmgr_dal_KDV.dll to C:\Program Files\Micro-Manager-1.3\
- (re)start Micro-manager

Information about the focus drive is available from http://www.kdvelectronics.eu/focusdrive/FocusDrive.html
Information about micro-manager is available from http://www.micro-manager.org
